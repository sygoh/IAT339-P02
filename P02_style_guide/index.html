Tabs N' Blanks Style Guide

A style guide for Tabs N' Blanks, a hypothetical store selling 3D woodcraft puzzles, for the purpose of a web design course project.

Created By: IAT339 D101 Winona Huang 301165181 & Joanne Goh 301160452

Go to: Interactive Elements, Text Elements, Combined Elements,

Navigation Button

CSS Selector:

.nav-menu, .nav-menu ul, .nav-menu ul li, .nav-menu ul li a

Sample Code:

<div class="nav-menu">
<ul>
<li><a href=""><span>Home</span></a></li>
</ul>
</div>

Rendered Element:

  • Home

In-text link

CSS Selector:

.in-text-link, .in-text-link a, .in-text-link a:hover

Sample Code:

<div class="in-text-link">
<p>This is a sample paragraph. <a href="">Visit our shop</a> to browse through the puzzle products we have available.</p>
</div>

Rendered Element:

This is a sample paragraph. Visit our shop to browse through the puzzle products we have available. Aenean euismod bibendum laoreet. Proin gravida dolor sit amet lacus accumsan et viverra justo commodo.
